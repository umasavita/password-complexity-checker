# 🔐 Password Complexity Checker

## 📌 Description
This project checks the strength of a password based on security criteria.

## 🛠 Tools Used
- Python
- Regex

## 🚀 Features
- Checks length, uppercase, lowercase, numbers, symbols
- Classifies password: Weak / Medium / Strong
- Provides suggestions

## ▶️ How to Run
python password_checker.py


## 📸 Example
Input: Uma123  
Output: Medium  

## 📚 Learning
- Regex usage
- Password security principles
